This manual describes the features of the R package CellTrails and how to utilize them. To view the handbook please view this project’s Pages site.

CellTrails is an extension for R (https://www.r-project.org), which is a free software environment for statistical computing and graphics. The CellTrails package can be installed by using devtools within an active R session.

devtools::install_github("dcellwanger/CellTrails")
